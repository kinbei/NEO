```
192.168.0.210 code.project1.local

```
注：  
(1) 如果遇到无法保存，请右键文件hosts并找到 `属性` -> `安全`，选择你登录的用户名，点击 编辑 ，勾选 `写入` 即可。

(3) `code.project1.local` 为内网测试服务器
